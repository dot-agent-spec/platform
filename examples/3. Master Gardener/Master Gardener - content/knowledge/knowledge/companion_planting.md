# Companion Planting Reference

## Beneficial Pairings

**Tomatoes** — basil (repels flies, improves flavor), marigolds (repels nematodes/aphids/whiteflies),
carrots (loosen soil), parsley (attracts beneficials), asparagus (tomatoes repel asparagus beetle)

**Peppers** — basil (repels aphids/spider mites), onions/garlic, carrots, spinach (living mulch),
tomatoes (similar needs)

**Beans** — corn (beans fix nitrogen; corn provides support), squash (Three Sisters),
carrots, cucumbers. **Avoid:** onions, garlic, fennel

**Brassicas** (cabbage, broccoli, kale) — dill (attracts beneficial wasps), mint (repels cabbage moths),
rosemary (repels cabbage fly), nasturtiums (trap crop for aphids), onions.
**Avoid:** strawberries, tomatoes, pole beans

**Carrots** — onions/leeks (repel carrot fly), lettuce (shade + space), tomatoes, rosemary/sage.
**Avoid:** dill (cross-pollination risk)

**Cucumbers** — beans (nitrogen), corn (shade), radishes (trap crop for cucumber beetles),
dill (pollinators), sunflowers. **Avoid:** potatoes, aromatic herbs

**Lettuce & Salad Greens** — carrots (different root depths), radishes (quick harvest),
strawberries, chives (repel aphids), tall plants (shade in summer)

## Pest-Repelling Plants

| Pest | Plants That Repel |
|---|---|
| Aphids | Garlic, chives, nasturtiums, marigolds, mint |
| Cabbage moths | Mint, rosemary, thyme, sage, hyssop |
| Carrot fly | Onions, leeks, rosemary, sage, wormwood |
| Cucumber beetles | Radishes, tansy, catnip |
| Japanese beetles | Garlic, chives, catnip, tansy |
| Squash bugs | Nasturtiums, tansy, radishes, marigolds |
| Tomato hornworm | Borage, marigolds, basil, dill |
| Whiteflies | Marigolds, nasturtiums, basil |

## Trap Crops

| Trap Crop | Lures Away |
|---|---|
| Nasturtiums | Aphids |
| Radishes | Cucumber beetles, flea beetles |
| Blue Hubbard squash | Squash bugs from other squash |
| Mustard | Cabbage pests from brassicas |

## Incompatible Combinations

| Keep Apart | Reason |
|---|---|
| Tomatoes + Brassicas | Compete for nutrients; stunt growth |
| Beans + Onion family | Onions inhibit bean growth |
| Fennel + Most vegetables | Fennel inhibits many plants |
| Potatoes + Tomatoes | Share diseases (blight) |
| Potatoes + Squash | Compete heavily; deplete soil |
| Corn + Tomatoes | Both heavy feeders; attract same worms |

## Planting Calendar (relative to last frost date)

| Weeks Before/After Last Frost | Start Indoors |
|---|---|
| 8–10 weeks before | Peppers, eggplant, onions (from seed), slow herbs (rosemary, oregano) |
| 6–8 weeks before | Tomatoes, basil, brassicas, lettuce (for transplant) |
| 4–6 weeks before | Cucumbers, squash, melons (short season), fast herbs (cilantro, dill) |

| Weeks Before/After Last Frost | Direct Sow Outdoors |
|---|---|
| 2–4 weeks before | Peas, spinach, lettuce, radishes, carrots (with protection), onion sets |
| After last frost | All warm-season crops; beans, corn, squash, cucumber, melon |

## Succession Planting

| Crop | Interval |
|---|---|
| Lettuce, spinach, radishes | Every 2 weeks |
| Beans, cucumbers | Every 3 weeks |
| Carrots, beets | Every 4 weeks |
| Corn | Twice per season (3 weeks apart) |
